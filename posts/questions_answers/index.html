SSH

Q1: ssh 连接提示：

1
2
3
4
5
6

Connecting to xx.xx.xx.xx:xx...
Connection established.
To escape to local shell, press 'Ctrl+Alt+]'.
Connection closing...Socket close.
Connection closed by foreign host.
Disconnected from remote host(pr) at 11:38:50.

A1: 可能是 SSH 服务器没设置保活时间间隔[1]，设置属性解释[2]

vim /etc/ssh/sshd_config

1
2
3

# 编辑以下两个属性, 并去掉行首的'#'
ClientAliveInterval 60
ClientAliveCountMax 3

到控制台重启 ssh

1
2

# /etc/init.d/ssh restart
systemctl restart sshd

网页

分别使用仓库对 githubpages 和 Netlify 搭建，域名使用 Cloudflare进行设置解析，遇到以下问题：

  • q1: 网页没有格式，css 无法进行网页的生成。查看网页 css 代码发现：

    Failed to find a valid digest in the 'integrity' attribute for resource 'https://xxxxx.ga/css/meme.min.fab688a4ce93176b6a089406d30086d60faeb496aae18e66623ee7b303123d53.css' with computed SHA-256 integrity'Df4VYwE3+jo4Qkh49Ym8EhhhZZhWuezgFWQ9pnUksAY='. The resource has been blocked.

    a1: 解决方法： 关闭 Cloudflare 的 css/js/html 数据压缩

  • q2: Netlify中使用 hugo 进行静态网页制作，其中对 theme 进行git submodule add订阅，异地进行数据更新维护时，发现本地无法进行预览，缺少模板文件（template、html 等等，提示需要进行模板文件的创建）

    a2: 本地的 clone 的主题文件夹为空，需要进行清理，然后重新订阅主题，才能正确进行预览
